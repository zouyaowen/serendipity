# Git 使用记录

## 文件基本操作

```shell
# 添加问价
git add file_name
# 提交文件记录
git commit -m '提交修改记录备注'
# 推送本地仓库至远程仓库
git push

# 添加并比较文件命令整合成一个命令
git commit -am "提交描述"
-a Tell the command to automatically stage files that have been modified and deleted, but new files you have not told Git about are not affected  修改的及删除的文件直接提交，新增的文件不受影响
-m Use the given as the commit message. If multiple -m options are given, their values are concatenated as separate paragraphs
-a  all  -m message
```
